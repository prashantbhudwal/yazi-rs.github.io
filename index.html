Yazi

⚡️ Blazing fast terminal file manager written in Rust, based on async I/O.

Get Started
Full Asynchronous Support

All I/O operations are asynchronous, CPU tasks are spread across multiple threads, making the most of available resources.

Async Task Scheduling and Management

Provides real-time progress updates, task cancellation, and internal task priority assignment.

Support for Multiple Image Protocols

Also integrated with Überzug++, covering almost all terminals.

Code Highlighting and Image Encoding

Combined with the pre-caching mechanism, greatly accelerates image and normal file loading.

The fastest way to work with files.

Scrollable Preview

Preview various types of files, and scroll the content of file while previewing.

Visual Mode and Bulk Rename

Bulk select files in visual mode, and rename them.

Vim-like Input and Select

Edit names quickly and choose open behavior using built-in input and select components.

Multi-Tab with fzf and zoxide

Navigate large directory trees quickly with multiple tabs and jump tools.

Multi-Select and Task Management

Queue file operations with task scheduling, progress tracking, and cancellation support.

Incremental Find

Find files as you type with instant match updates and clear position feedback.

Search

Search by name with fd, by content with rg, then act on results from the same flow.

Quick install (macOS)
Install Yazi and common optional dependencies.
brew install yazi ffmpeg sevenzip jq poppler fd ripgrep fzf zoxide resvg imagemagick
